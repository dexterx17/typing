# Typing CSS

Practica de CSS copiando el excenlete estilo de typing.com

## Herramientas

-   [https://tailwindcss.com/docs/installation](TailwindCSS) utility-first CSS framework


## Instalacion


``` npm install ```

``` npx tailwindcss -i src/tailwind.css -o css/app.css ```
